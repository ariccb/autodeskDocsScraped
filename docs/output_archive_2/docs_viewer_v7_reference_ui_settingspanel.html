Autodesk.Viewing.UI

SettingsPanel

Extends Autodesk.Viewing.UI.SettingsPanel

new SettingsPanel(parentContainer, id, title, options)

UI panel specifically designed for application settings.

The user can add new options to each of the tabs.

Parameters

Expand all
parentContainer*
HTMLElement
The container for this panel.
id*
string
The id to assign this panel.
title*
string
The title of this panel.
options
object
An optional dictionary of options.
width
number
Override panel’s minimum width
heightAdjustment
number
Override panel’s extra content height, to account for non-scrolling elements.
