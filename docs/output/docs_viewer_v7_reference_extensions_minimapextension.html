Autodesk.Viewing.Extensions

MinimapExtension

Provides a 2d Minimap to show the view of the current document.

The extension id is: Autodesk.BIM360.Minimap

new MinimapExtension()

Examples

viewer.loadExtension('Autodesk.BIM360.Minimap')
